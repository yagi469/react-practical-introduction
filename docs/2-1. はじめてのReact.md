それでは早速、Create React App を利用して、はじめての React アプリを作成してみましょう。本節では、作成から実行までの手順を押さえるとともに、自動生成されたコードを読み解くことで、React アプリの外枠を理解していきます。

## 2-1-1. Reactアプリの作成

まずは、Create React App によるアプリの作成～実行の手順からです。

[1] プロジェクトを作成する

React アプリを開発するには、まず、開発の土台（場）となる**プロジェクト**を作成する必要があります。これには、コマンドラインから以下のコマンドを実行してください。プロジェクトを作成する先は自由に選択して構いませんが、本書では「c:\data」配下に作成するものとします。

```bash
# カレントフォルダーを移動
> cd c:\data

# アプリを作成
> npx create-react-app my-react
Need to install the following packages:
  create-react-app@5.0.1
Ok to proceed? (y)

Creating a new React app in C:\data\my-react.

Installing packages. This might take a couple of minutes.
Installing react, react-dom, and react-scripts with cra-template...
...中略...
We suggest that you begin by typing:

  cd my-react
  npm start

Happy hacking!
```

`npx` は Node.js で提供されるコマンドの一種で、ローカルにインストールされたツールを実行するためのパッケージランナーです。この例であれば、`create-react-app` を実行し、`my-react` という名前でアプリ（プロジェクト）を作成しなさい、という意味になります。

ただし、初回の実行ではそもそも `create-react-app` がインストールされていないので、不測のパッケージをインストールしてもよいかと訊かれているわけです。「`y`」を入力し、先に進みましょう[^1]。

[^1]: 当然、次回以降の実行では表示されなくなるはずです。

[2] プロジェクトの内容を確認する

「c:\data\my-react」というフォルダーが生成されているので、VSCodeで開いておきましょう。これにはエクスプローラーから /my-react フォルダーを右クリックし、表示されたコンテキストメニューから [その他のオプションを表示] - [Codeで開く][^2]を選択してください。

[^2]: [Codeで開く]メニューは、VSCodeのインストール時に[エクスプローラーのディレクトリコンテキストメニューに[Codeで開く]アクションを追加する]をチェックした場合に表示されます。意図したメニューが表示されない場合は、今一度、1-2-3項の手順を確認してください。

VSCode で my-react プロジェクトが開くので[^3]、作成直後のプロジェクト構造を確認してみましょう。[エクスプローラー]ペインから、以下のようにフォルダーやファイルが生成されていることを確認してください。

[^3]: [このフォルダー内のファイルの作成者を信頼しますか？] というダイアログが表示される場合には、そのまま [はい、作成者を信頼します] をクリックしてください。

├── public/
│     └── favicon.ico.............ファビコン

│     └── index.js................起動ファイル
